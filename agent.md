Данный проект является fsd с тайпскрипт, ты должен разделять бизнес логику и ui логику, не городить в ui компоненты логику и наоборот не городить в функциональные компоненты ui, желательно максимальное упрощение. Для логики создавать кастомные хуки, придерживайся чистого кода. Всегда используй StateProvider для обработки isLoading/isError/empty вместо ручных проверок и прямого рендера LoadingState/ErrorState.
